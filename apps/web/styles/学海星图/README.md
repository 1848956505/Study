# 学海星图 · Stars & Atlas

> 深空 + 星光 + 紫青双色 + 数字大号
> 适合:多模块扩展 / 个人宇宙感 / 信息密度大

## 视觉关键词

- **深空背景** — `#050813` 最深空,13 颗多色 radial 模拟星点
- **星光闪烁** — 标题右上角 12px 暖星黄圆点 + `box-shadow + 闪烁动画`
- **紫青双色** — 星蓝 `#5b8def`(主) + 星紫 `#9d7cff`(辅),双色穿插
- **暖星黄** — `#ffd180` 仅用于"已选中"标记、tab 指示、状态徽章
- **数字大号** — H1 42px,font-variant-numeric tabular-nums 数字等宽
- **信息密度** — 间距偏小(基础 3/6/10/14),适合塞更多内容

## 5 项视觉特色

| # | 特色 | 体现 |
| --- | --- | --- |
| ① | 13 颗多色星点 | 多层 radial-gradient 模拟 3 种星色 + 4 种星大小 |
| ② | 星光闪烁 | 标题 mark 右上角暖星黄 + 2s 闪烁动画 |
| ③ | 紫青双色 | 主色蓝 + 辅色紫,选中态混合 glow |
| ④ | 数字大号 | 42px H1 + 24px H2 + 数字 tabular-nums |
| ⑤ | 高密度排版 | 间距 3/6/10/14,卡片紧凑 |

## 适合场景

- ✅ 多模块信息密集(每个模块都能塞满数据)
- ✅ 长期个人空间(深空感带来"专属宇宙"归属感)
- ✅ 数据可视化(数字大号 + tabular-nums 适合图表)
- ❌ 长文阅读(深色 + 紧凑不利于阅读节奏)
- ❌ 暖意需求(深空感偏冷)
- ❌ 浅色工作流(整页深色不适应)
