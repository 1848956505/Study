# 光年玻璃 · Glass & Glow

> 深空紫蓝 + 玻璃拟态 + 霓虹微光 + 现代 sans
> 适合:多模块扩展 / 强科技感 / 暗色工作流

## 视觉关键词

- **深空背景** — `#0a0e1f` → `#1a1f3a` 渐变 + 多层 radial 星点
- **玻璃拟态** — 卡片 `rgba(255, 255, 255, 0.04~0.12)` + `backdrop-filter: blur(20px)`
- **紫蓝主色** — `#7c5cff`(主强调),与青绿 `#00d4a8`(成功/活力)形成冷暖对比
- **霓虹微光** — 选中态用 `box-shadow: 0 0 24px rgba(124, 92, 255, 0.45)`
- **现代 sans** — 全部 `Inter / SF Pro`,**无衬线**

## 5 项视觉特色

| # | 特色 | 体现 |
| --- | --- | --- |
| ① | 玻璃拟态卡片 | 顶部 1px 紫蓝光带作分隔,模糊背景透出星点 |
| ② | 紫蓝渐变光带 | 区块标题 / 卡片 / 标题字色 |
| ③ | 霓虹微光边框 | focus / 选中态带 glow 阴影 |
| ④ | 青绿状态点 | 笔记保存状态、tab 标记、强调按钮 |
| ⑤ | 现代 sans 大字 | H1/H2 不用衬线,数字大号,字距收紧 |

## 适合场景

- ✅ 长时间工作(暗色护眼)
- ✅ 信息密度大(玻璃层叠仍清晰)
- ✅ 多模块共存(强视觉语言,各模块特征鲜明)
- ❌ 短时阅读(冷色易疲劳)
- ❌ 打印/截图(深色背景不利于文档分享)
