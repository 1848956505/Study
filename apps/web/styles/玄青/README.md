# 玄青 · Xuanqing (Modern)

> 现代化设计 · 玄青主色延续 · 去除所有古风装饰
> 保留"修仙"概念性(玄青主色 / 灵气紫 / 克制),但用 Linear / Vercel / Notion 的现代语言表达

## 为什么是"现代化玄青"而不是"古风仙府云笺"

- **去掉**:楷书、隶书、宋体、朱砂印章、竹简纤维、卷云、米黄底色、篆体字、"道"字底纹、隶书标签
- **保留**:玄青主色(深蓝,品牌延续) + 灵紫渐变(现代紫蓝) + 翠绿(状态) + 琥珀金(强调)
- **新增**:Inter 字体 / 渐变光带 / 干净阴影 / Inter font-variant-numeric tabular-nums / neumorphism 极简

## 视觉关键词

- **玄青主调** — `#1a3a5c` 深蓝(非古风青绿),冷调现代
- **灵紫渐变** — `#7c5cff` 紫蓝,主按钮用 `xuanqing → lingzi` 渐变,选中态底部细光带
- **琥珀金** — `#e5a55c`,强调色,状态徽章
- **翠绿** — `#3eb489`,成功/已掌握
- **中性灰底** — `#f5f6f8`,非米黄,无装饰纹理
- **Inter 字体** — 全部现代 sans,数字 `font-variant-numeric: tabular-nums`
- **细线 + 轻阴影** — `--shadow-paper: 0 1px 3px ...`,Linear 风格

## 5 项视觉特色

| # | 特色 | 体现 | 现代映射 |
| --- | --- | --- | --- |
| ① | **玄青主调** | 玄青深蓝 + 中性灰底,无装饰纹理 | Linear / Vercel 极简 |
| ② | **灵光选中** | 选中态底部 2px 灵紫渐变光带 + 灵紫软晕 | 现代 Tab Bar 指示 |
| ③ | **现代大标题** | Inter 32px,字距 -0.02em,数字大号紧凑 | Apple / Stripe 大标题 |
| ④ | **克制留白** | 行高 1.55,8 基准网格,`max-width: 72ch` | 内容中心阅读 |
| ⑤ | **紫青光晕** | focus 用 3px 灵紫 soft + 1px 实线 | 现代 focus 环 |

## 应用外壳特色

- **rail 现代图标** — 32px 圆角矩形,选中态灵紫软晕
- **topbar 干净细线** — 搜索框 focus 时切换为白色 + 灵紫边 + 光晕
- **tab 底部渐变光带** — 紫→琥珀渐变,2px 高,圆角顶端
- **tab close 悬停显隐** — 鼠标悬停时显示 × 按钮(Linear / Vercel 风格)
- **状态栏数字等宽** — `font-variant-numeric: tabular-nums`

## 适合场景

- ✅ 长期个人专业工具
- ✅ 多模块扩展(留白 + 渐变可适配)
- ✅ 现代开发者/设计者审美
- ✅ 暗色 / 浅色 / 高对比度主题切换
- ❌ 强文化场景(古风 / 仪式 / 游戏化养成)
- ❌ 印刷品/海报类(现代化不利于排版)
