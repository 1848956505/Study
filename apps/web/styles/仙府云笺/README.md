# 仙府云笺 · Xianfu Cloud Paper

> 玄青主色 + 朱砂印章 + 行楷大字 + 卷云分隔 + 灵气焦点
> 专为"学习加速器 + 修仙录"主题打造,道家东方意境与游戏化养成的融合

## 视觉关键词

- **玄青主色** — `#2d4a52`(道家青),深邃克制,呼应"道法自然"
- **朱砂印章** — `#a83838`(红),选中态用 24×24 红方印(内含篆体字"录""经"),微旋转 3° 模拟"刚盖上"的仪式感
- **灵紫点缀** — `#6a4a8a`,灵气 / 紫府 / 玄妙,用于 focus 环 / 标题装饰
- **道金尊崇** — `#c8a868`,师尊 / 阶段成就 / 高光强调
- **墨翠炼丹** — `#3a6a52`,成功 / 灵草 / 已掌握
- **仙纸底色** — `#ede5cf`(竹简米黄) + 极淡竖向纤维纹(90° repeating-linear-gradient)
- **行楷大字** — H1/H2 用 `STKaiti / Kaiti` 行楷,带笔锋,模拟修真卷轴
- **隶书区块** — section 标题用 `STLiti` 隶书,带篆刻感

## 5 项视觉特色

| # | 特色 | 体现 | 修仙映射 |
| --- | --- | --- | --- |
| ① | **竹简纸面** | 米黄底 + 极淡竖向纤维,模拟竹简质感 | "修真以竹简载道" |
| ② | **朱砂印章** | 选中态 24×24 红方印(篆体字"录"),微旋转 | "盖印封册,正式纳入修炼" |
| ③ | **行楷大字** | H1 用行楷 32px,带笔锋,字距 0.1em | "卷轴大字,以示郑重" |
| ④ | **卷云分隔** | section 之间用卷云纹(☁)居中分隔,行高 1.7 | "云气氤氲,隔而不绝" |
| ⑤ | **灵气焦点** | 输入聚焦时,紫青光晕扩散,模拟"指尖灵气散开" | "聚气凝神,元神入定" |

## 应用外壳特色

- **rail 圆形篆体** — 选中态用朱砂竖条 2px
- **topbar 木纹底** — 底部渐变条(玄青→道金→朱砂)
- **tab 卷轴挂签** — 选中 tab 出现朱砂"经"字印
- **状态栏篆体装饰** — 顶部三色细线,文字用隶书
- **空状态卷云** — 顶部 ☁ 卷云符号

## 字体分级

| 用途 | 字体 | 场景 |
| --- | --- | --- |
| H1 / H2 / 笔记标题 | `STKaiti` 行楷 | 大标题、笔记标题、强调 |
| 区块标题 / 标签 | `STLiti` 隶书 | 卡片标题、tab 文字、按钮文字 |
| 正文 / UI 控件 | `PingFang SC` 无衬线 | 段落、输入、菜单 |
| 代码 / 数字 | `Consolas` 等宽 | 代码块、统计数字 |

## 适合场景

- ✅ 长期个人修炼项目(慢积累,与修炼节奏契合)
- ✅ 知识沉淀类工具(笔记/复习/总结)
- ✅ 与"修仙录"模块深度契合(任务游戏化)
- ✅ 东方文化爱好者
- ❌ 多模块高密度场景(行高/留白偏松,信息密度低)
- ❌ 短时快速操作(仪式感强 = 慢节奏)
- ❌ 现代科技感偏好(整体偏古风)
