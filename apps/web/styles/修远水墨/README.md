# 修远水墨 · Ink & Mist

> 纸白 + 深青墨黑 + 朱砂印章 + 宋体大字
> 适合:东方禅意 / 慢思考 / 长期个人项目

## 视觉关键词

- **纸白底色** — `#f5f3ee` + 极淡 45° 墨纹 + 中心墨晕
- **墨色阶** — 从 `#8b9094` 浅墨到 `#0a0d0e` 浓墨,5 档递进
- **深青主色** — `#2d4a4a` 主品牌,克制内敛
- **朱砂点缀** — `#b23a3a` 仅用于:印章/竖条选中/标题装饰/危险操作
- **宋体大字** — H1/H2/品牌用 `STZhongsong/Songti`,正文用无衬线
- **大量留白** — `--line-height-base: 1.7`(比其他风格更松)

## 5 项视觉特色

| # | 特色 | 体现 |
| --- | --- | --- |
| ① | 极淡纸纹 | `repeating-linear-gradient` 45° 极淡青墨线 |
| ② | 朱砂印章 | 选中态 3px 宽朱砂竖条 + 背景朱砂软晕 |
| ③ | 宋体大字 | H1 字距 0.15em,行高 1.3,带强烈书页感 |
| ④ | 强化留白 | 行高 1.7、max-width 64ch(比晨光窄)、padding 更大 |
| ⑤ | 墨色焦点 | focus 环用墨色 2px 实线 + 4px 间隙,不刺眼 |

## 适合场景

- ✅ 个人长期项目(慢积累,知识沉淀)
- ✅ 文本为主的工作流(阅读、写作、思考)
- ✅ 与"修仙录"主题深度契合(东方意境)
- ❌ 信息密度高场景(留白多 = 占空间)
- ❌ 多模块共存(视觉语言偏单一)
- ❌ 短时快速操作(慢节奏不匹配)
